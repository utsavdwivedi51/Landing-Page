l# SecureVPN Landing Page

A modern, responsive landing page for a VPN service (“SecureVPN”), inspired by ExpressVPN’s marketing page.  
Built with **pure HTML and CSS** (no frameworks) to practice layout, typography, and component-based design.

Live Demo: https://utsavdwivedi51.github.io/Landing-Page/

## Overview

This project is a single-page marketing site that showcases:

- A hero section with call-to-action
- “Secure access” feature highlight
- “One click to a safer internet” section with custom illustration
- Multi-device support section
- Three key feature cards
- Global server coverage map + stats
- Simple footer

It’s ideal for learning responsive web design and can be reused as a template for other SaaS / product landing pages.

---

## Tech Stack

- **HTML5**
- **CSS3**
- (Optional) **JavaScript** – empty file included for future interactions

No external CSS frameworks are used. Only Google Fonts are loaded via CDN.
